# 🏃 Тренажер Утренней Зарядки для Детей

Мобильное приложение для Android, которое помогает детям заниматься утренней гимнастикой с весёлым интерфейсом и настраиваемыми упражнениями.

## 📱 Возможности

✅ **3 встроенных упражнения:**
- Приседания 🦵
- Прыжки 🤸
- Отжимания 💪
- Наклоны 🧘
- Бег на месте 🏃

✅ **Полностью настраивается:**
- Количество упражнений (1-5)
- Количество повторений в подходе (1-20)
- Количество подходов (1-5)
- Время отдыха между подходами (5-60 сек)

✅ **Удобный интерфейс:**
- Большие эмодзи для визуализации упражнений
- Яркие, привлекательные цвета
- Четкие инструкции для каждого упражнения
- Таймер отдыха
- Отслеживание прогресса

## 🛠️ Технические требования

- Android 7.0+ (API 24+)
- Kotlin
- Jetpack Compose
- Android Studio

## 📥 Установка и сборка

### Требования:
- Android Studio 2022.1 или выше
- Java 11 или выше
- Android SDK 34

### Шаги:

1. **Клонируйт�� репозиторий:**
```bash
git clone https://github.com/robboclublahtateachers-art/morning-exercises-app.git
cd morning-exercises-app
```

2. **Откройте проект в Android Studio:**
   - File → Open → Выберите папку проекта
   - Android Studio автоматически загрузит зависимости

3. **Соберите проект:**
   - Build → Make Project (Ctrl+F9 или Cmd+F9)

4. **Создайте APK:**
   - Build → Generate Signed Bundle/APK
   - Следуйте инструкциям мастера
   - APK файл будет сохранен в `app/release/`

5. **Установите на устройство:**
   - Подключите Android устройство
   - Перетащите APK на устройство или используйте ADB:
   ```bash
   adb install app/release/app-release.apk
   ```

## 🎮 Использование

1. **Главный экран:**
   - Нажмите "Начать тренировку" для старта
   - Нажмите "Настройки" для изменения параметров

2. **Экран настроек:**
   - Используйте ползунки для выбора параметров
   - Нажмите "Сохранить" для применения

3. **Экран упражнения:**
   - Смотрите на большой эмодзи и описание упражнения
   - Следуйте инструкциям
   - Нажимайте "Отдыхать →" после каждого повтора
   - После отдыха автоматически появится счетчик

## 🐛 Решение проблем

### Проблема: Android Studio не видит SDK
**Решение:**
- File → Project Structure → SDK Location
- Установите Android SDK (API 34)

### Проблема: Ошибка при сборке
**Решение:**
- Build → Clean Project
- Build → Rebuild Project

### Проблема: APK не устанавливается
**Решение:**
- Убедитесь, что на устройстве включена установка из неизвестных источников
- Settings → Security → Unknown Sources (ON)
